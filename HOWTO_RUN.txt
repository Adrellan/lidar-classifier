Egyszerű futtatás (CUDA-s környezet feltételezve)

1) Aktiváld a virtuális környezetet  
   source .venv/bin/activate

2) Tedd a nyers adatot a train_data alá  
   - LAS/LAZ: train_data/las_laz  
   - PCD: train_data/pcd  (ha itt nincs .las/.laz, a script konvertálja)

3) Állítsd be a tanítási paramétereket a config.yaml-ben  
   - Tanulási paraméterek: pipeline.optimizer.lr, batch_size, max_epoch, stb.  
   - Csempézés: preprocess.tile_size_m, max_points_per_tile, train/val arányok

4) Indítsd a tanítást  
   - LAS/LAZ: python scripts/train.py --las  
   - PCD:     python scripts/train.py --pcd

Automatikusan történik
- PCD módban, ha nincs .las/.laz a train_data/pcd-ben, PDAL-lal konvertál.  
- Újracsempéz minden futtatáskor a runs/<las|pcd>/tiles mappába.  
- Logok és checkpointok: runs/<las|pcd>/logs, cache: runs/<las|pcd>/cache.

Előfeltételek
- PDAL CLI szükséges a PCD->LAS konverzióhoz.  
- GPU-s futtatáshoz CUDA-s PyTorch + Open3D-ML legyen telepítve.
